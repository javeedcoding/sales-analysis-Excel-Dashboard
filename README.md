# Sales Analysis Excel Dashboard

## Overview

This project presents a Sales Analysis Dashboard created using Microsoft Excel. It analyzes sales performance across countries, product lines, months, deal sizes, and order status using Pivot Tables, Pivot Charts, KPI Cards, and business insights.

## Dashboard Features

- Total Sales KPI
- Average Sales KPI
- Highest Sales KPI
- Total Orders KPI
- Sales by Country
- Sales by Product Line
- Sales by Month
- Sales by Deal Size
- Sales by Status
- Business Insights

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas
- Dashboard Design

## Key Insights

- USA generated the highest sales.
- Classic Cars was the highest-performing product line.
- November recorded the highest monthly sales.
- Medium deal size generated the highest revenue.
- Shipped orders contributed the highest sales.
